## Libraries
- Retrofit: Type-safe HTTP client for Android that simplifies API calls by mapping JSON responses to Java/Kotlin objects.
- OkHttp: HTTP client for Android that manages network requests, caching, and connection.
- Gson: Converts JSON strings to Java/Kotlin objects and vice versa.
- Firebase: To integrate the application with authentication and bookmark database for each user.
- CameraX: Library that simplifies camera integrations.
- Glide: Library for efficient image loading, caching, and displaying.
